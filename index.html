A Parametric Task Space for Open-World Learning: From Task Evaluation to Generation

Chen He, Hui Nie, Bin Zhang, Fengyuan Yang, Ruiping Wang, Xilin Chen
Key Laboratory of AI Safety of CAS, Institute of Computing Technology,
Chinese Academy of Sciences(CAS), Beijing, China
Paper Supplementary Code arXiv
-->

An illustration of the task evaluation and generation framework for Open World Learning: (1) Task evaluation: By using the carefully designed difficulty metrics across the five dimensions of openness, a predefined task can be mapped to a point in the parametric task space spanned by them, as visualized in the radar chart; (2) Task generation: By sampling from the rich data source provided by the data generation engine, our framework can create task-specific datasets with varying degrees of openness and difficulty that meet the user requirements.

Abstract

Open-world learning needs to contend with multiple openness dimensions—such as open-set, zero-shot, few-shot, long-tailed, and incremental learning—that routinely co-occur in real-world scenarios. However, existing task formulations predominantly address these dimensions in isolation, resulting in evaluations of limited breadth (single-dimension) and insufficient depth (single difficulty level per dimension). In this work, we introduce a parametric task space that unifies multiple openness dimensions along multi-axis parameters. Under this space, an open-world task is uniquely positioned by five parameters that quantify open-set degree, transfer difficulty, sample scarcity, imbalance degree, and old-class discard ratio. Conversely, given any target configuration in the space, our framework generates the corresponding dataset and evaluation splits on demand, enabling configurable open-world tasks with precise control over both breadth and depth. Experiments in object recognition and detection reveal consistent performance trends that generally align with our parameterized dimensions, underscoring the framework’s utility. The proposed task space provides a scalable foundation for holistic open-world learning research.

A comparative visualization of difficulty metrics across five openness dimensions. For example, "open set degree" is correlated with the ratio of unknown categories. As the "imbalance degree" intensifies, the distribution of instance numbers across the dataset becomes increasingly uneven, with many categories in extreme cases having only a few samples, thus elevating learning difficulty.

The pipeline of task generation for Open-World Learning comprises a meta-dataset generation and a task-specific dataset sampling stage. For the former stage, a meta-dataset is constructed, which enhances the original simulator with additional capabilities, including expanding object assets, adding thousands of textures, randomizing object poses, and introducing varied lighting conditions. For the latter stage, the meta-dataset is sampled to create task-specific (i.e., any combination of openness dimensions) datasets with varying levels of difficulty, by controlling the values of five core openness dimensions metrics.

Interactive interface for visualizing and generating tasks based on five category-related openness dimensions. Each slider controls a normalized difficulty metric, allowing users to explore how different combinations influence task characteristics.
